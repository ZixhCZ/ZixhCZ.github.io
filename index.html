    Hledáš Minecraft server? Nejnovější verzi? Nudí tě obyčejná Vanilla? Tak přestaň už hledat. Server je vytvořen na způsob Semi-Vanilla. Zde najdeš několik možností aktivity, kterou můžeš vykonávat buď sám nebo s ostatními spoluhráči, ale ať už si zvolíš možnost první či druhou určitě tě to nepřestane bavit a jentak pro tebe tento server neskončí, protože má v zásobě spoustu nápadů, co se budou realizovat postupem času. A čím je jiný od ostatních serverů? Server je založen na kvalitním hostingu se svou doménou. Server bude mít téma, který se nazývá EVOLVE.
